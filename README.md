Boil
====

Boil is an opinionated interactive command to easily setup the boilerplate of your next frontend.

# Usage

<!-- usage -->
```sh-session
$ npm install -g boil
$ boil COMMAND
running command...
$ boil (-v|--version|version)
boil/0.0.0 win32-x64 node-v14.15.1
$ boil --help [COMMAND]
USAGE
  $ boil COMMAND
...
```
